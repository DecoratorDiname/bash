# Набор полезных скриптов для Bash

Этот набор содержит простые скрипты на Bash, демонстрирующие базовые задачи:
- приветствие по имени
- калькулятор суммы
- проверка чётности
- создание структуры проекта
- подсчёт строк в файле
- генерация пароля
- поиск файлов по расширению

Содержание
- приветствие по имени
- сумма
- чётность
- структура проекта
- счётчик строк
- генератор паролей
- поиск файлов
- инструкций по запуску

Примечание
- Все скрипты совместимы с bash.
- Для запуска достаточно дать права на выполнение и запустить файл.
